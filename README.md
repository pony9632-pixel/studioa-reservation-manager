# StudioA 門市預約管理（桌面程式）

介接 StudioA 門市預約後台（`shopcms`）的小工具，提供四個分頁：

1. **預約總覽** — 目前剩餘「已預約」數、各狀態統計、**型號統計**、**會員等級統計**
2. **區間查詢** — 指定日期區間（含「最近 7 天」等快捷鈕），看新增預約筆數、各狀態與每日分佈
3. **狀態管理** — 輸入**預約單號查單**，或用日期區間列出，並**變更狀態**（已預約／已配貨／已到貨／保留／已取貨／放棄／取消）
4. **變更紀錄** — 自動偵測並列出狀態變更（含後台那邊直接改的，例如放棄／取消）；從開始使用後累積

> 資料直接來自你們後台的 API，登入帳號即你平常登入後台的帳號。詳見 [api_notes.md](api_notes.md)。

---

## 一、安裝

需要 Python 3.9 以上（建議 3.11+）。

```bash
cd 預約後台
python3 -m pip install -r requirements.txt
```

## 二、執行

```bash
python3 app.py
```

啟動後會跳出登入視窗，輸入後台帳號（email）與密碼即可。可勾選「記住帳號」方便下次使用。

## 三、各分頁說明

### 預約總覽
- 選擇「預約日期區間」後按 **查詢 / 重新整理**。
- 上方卡片是後台直接回傳的統計：總數量、**目前剩餘已預約**（藍色卡）、已配貨、已到貨、保留、已取貨、放棄、取消、取貨率。
- 下方「型號統計／會員等級統計」可切換範圍：**僅已預約（剩餘）**、未取貨、全部。

### 區間查詢
- 按快捷鈕（今天／最近 7 天／最近 30 天／本月）或自選區間，再按 **查詢**。
- 顯示該區間「新增預約」總筆數，並列出各狀態、每日新增筆數。

### 狀態管理
- 在「預約單號」欄輸入單號按 Enter（或「查單」）即可查到該筆。
- 不知道單號時，可用「日期區間 + 狀態」按 **列出** 瀏覽。
- 載入清單後，可用上方**篩選**快速縮小：先按 **大分類**（Mac／iPhone／iPad／AirPods），再用 **活動／梯次／會員等級／狀態** 下拉細篩，右側會顯示「符合 N／共 M 筆」。
- 在表格最左欄「✓」**勾選**要處理的單據（可多筆，或用「全選／全不選」），再按送出變更或列印標籤。
- 選取一或多列後按 **🖨 列印標籤**，把選取的（通常是「已到貨」）預約輸出成標籤 PDF（可選 Mac／iPhone 兩種版型），自動開啟後按 Cmd+P 列印。首次列印會自動安裝 `reportlab`。
- 在表格中**點選一或多列**（可按住 Cmd / Shift 多選），選擇要變更的狀態，按 **送出變更**，確認後即更新後台。
  - 門市標準可改：**已到貨／保留／已取貨**；其他狀態若後台不允許，會顯示後台回傳的錯誤訊息。

## 四、打包成單一執行檔（給沒有 Python 的同事用）

```bash
python3 -m pip install pyinstaller
pyinstaller --noconsole --onefile --name "StudioA預約管理" app.py
```

完成後執行檔在 `dist/` 目錄。macOS 第一次開啟若被攔，於「系統設定 → 隱私權與安全性」按「仍要打開」。

## 五、帳密儲存說明（請留意）

- 設定檔位置：`~/.studioa_reservation_app.json`
- 「記住密碼」會把密碼以 **base64 編碼**（**僅防止一眼看到，並非加密**）存在你自己的電腦。
- **請勿在公用電腦勾選「記住密碼」**；公用電腦只勾「記住帳號」即可。

## 六、疑難排解

| 狀況 | 處理 |
|---|---|
| 登入失敗：帳號或密碼可能錯誤 | 確認帳密；用瀏覽器登入後台確認帳號可用 |
| 未授權 / 請重新登入 | token 過期，用選單「帳號 → 重新登入」 |
| 連線失敗 | 檢查網路；確認可連到 studioa.com.tw |
| 某狀態改不動 | 後台可能僅允許門市改「已到貨/保留/已取貨」，其餘狀態由其他流程設定 |

## 七、注意事項

- 這是**內部工具**，會讀取客戶個資並可直接變更後台狀態，請勿外流原始碼或執行檔。
- 「送出變更」會**真的更新後台資料**，操作前會跳出確認視窗。
